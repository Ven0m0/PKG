# dolphin-mpn

## Description

Dolphin emulator fork optimized specifically for Mario Party Netplay.

## Source

- Upstream: <https://github.com/MarioPartyNetplay/Dolphin-MPN>
- AUR Package: <https://aur.archlinux.org/packages/dolphin-mpn>

## Build Instructions

```bash
cd dolphin-mpn
makepkg -si
```
